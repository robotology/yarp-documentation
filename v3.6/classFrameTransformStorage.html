Detailed Description

frameTransformStorage: A class that keeps a FrameTransformContainer updated by receiving transformations through the yarp::dev::IFrameTransformStorageSet interface and periodically querying a set of attached devices that expose the yarp::dev::IFrameTransformStorageGet interface.

It also exposes the yarp::dev::IFrameTransformStorageGet interface to allow stored transformations retrieval
