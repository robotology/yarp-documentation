Binary releases

Windows

Remarks
Important: to develop code using precompiled binaries packages have to match the compiler you use, make sure you download and install the correct version (i.e. MSVC 12 for Visual Studio 2013, etc..). If in doubt about the version number of your copy of Visual Studio, find the internal version number shown for it in this version history table.

Installers:

MSVC 2015/2017
Win32 yarp_3.1.0_v14_x86_1.exe
x64 yarp_3.1.0_v14_x86_amd64_1.exe

Linux (Debian/Ubuntu)

The easiest way to install YARP from binaries is through aptitude as explained in Installation on GNU/Linux.

Individual .deb packages can also be downloaded from GitHub

macOS

The easiest way to install YARP from binaries is through brew as explained in Installing YARP on macOS.

Brew bottles can also be downloaded from GitHub
