Detailed Description

fakeMicrophone : fake microphone device implementing the IAudioGrabberSound interface to generate a test sound.

It can generate various signals, i.e. sine, sawtooth, square wave, constant. This device driver derives from AudioRecorderDeviceBase base class. Please check its documentation for additional details.

Parameters required by this device are shown in class: FakeMicrophone_ParamsParser This device also inherits some parameters from AudioRecorderDeviceBase

See Audio in YARP for additional documentation on YARP audio.
