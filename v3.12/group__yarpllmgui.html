yarpllmgui: Controls a LLM device
Yarp GUIs

Controls and displays the conversation with a LLM device.

Collaboration diagram for yarpllmgui: Controls a LLM device:

Controls and displays the conversation with a LLM device.

Description

The gui uses a LLM_nwc_yarp device to connect to a remote LLM_nws_yarp device.

Parameters

remote_rpc          Mandatory.   Name of the rpc port opened by the nws.
streaming_port      Optional.    Name of the streaming port opened by the nws (by default /llm/conv:o).
