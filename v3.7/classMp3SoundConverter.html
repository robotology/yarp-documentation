Detailed Description

sound_compression_mp3: A portmonitor which is able to convert apply mp3 compression/decompression to yarp::sig::Sound data transmitted over the network.

Example usage: yarp connect /src /dest tcp+recv.portmonitor+type.dll+file.sound_compression_mp3

See Audio in YARP for additional documentation on YARP audio.
