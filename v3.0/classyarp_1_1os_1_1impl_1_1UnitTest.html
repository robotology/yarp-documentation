◆ heapMonitorBegin()

void UnitTest::heapMonitorBegin ( bool  expectAllocations = true)

Begin monitoring heap activity, specifically calls to new/new[]/delete/delete[].

It is important to eventually call heapMonitorEnd() to stop monitoring heap activity. You must guarantee that no heap activity is occurring in any other threads when you call this method.

Parameters
expectAllocationsset to true if heap activity is expected, set to false if heap activity is unexpected (stack traces will be shown for every new/delete in this case)
