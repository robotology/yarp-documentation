Detailed Description

Rangefinder2DTransformer: A device which acts a virtual laser, it attaches to another lidar, which provides a stream a measurements, and relocates them in the space.

The new origin can be specified both manually, as a 2D point in space, or though a transform.

Parameters required by this device are shown in class: Rangefinder2DTransformer_ParamsParser
