Detailed Description

This class is the parameters parser for class Odometry2D_nws_ros2.

These are the used parameters:

Group name Parameter name Type Units Default Value Required Description Notes
- period double s 0.02 0 refresh period of the broadcasted values in s -
- node_name string - - 1 name of the ros2 node -
- topic_name string - - 1 name of the topic where the device must publish the data must begin with an initial '/'
- odom_frame string - - 1 name of the reference frame for odometry -
- base_frame string - - 1 name of the base frame for odometry -

The device can be launched by yarpdev using one of the following examples (with and without all optional parameters):

yarpdev --device odometry2D_nws_ros2 --period 0.02 --node_name <mandatory_value> --topic_name <mandatory_value> --odom_frame <mandatory_value> --base_frame <mandatory_value>
yarpdev --device odometry2D_nws_ros2 --node_name <mandatory_value> --topic_name <mandatory_value> --odom_frame <mandatory_value> --base_frame <mandatory_value>
